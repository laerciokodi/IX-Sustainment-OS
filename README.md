## 📋 About IX-Sustainment-OS

IX-Sustainment-OS is a tool designed to help maintenance teams manage their work more clearly. It shows what can be fixed now, what is blocked, and why. It keeps technical information controlled and offers AI recommendations that need approval before use. The system logs all actions for review. It supports teams in deciding how to handle parts bottlenecks and technical data during maintenance.  

This program runs as a simple app on Windows, offering a clear interface without unnecessary features. It aims to keep workflows organized and transparent.

## 🖥️ System Requirements

To run IX-Sustainment-OS on Windows, your computer should meet these minimum requirements:

- Windows 10 or newer (64-bit recommended)  
- 4 GB of RAM or more  
- At least 200 MB of free disk space  
- Internet connection for initial download and updates  
- Microsoft .NET Framework 4.8 or higher (usually pre-installed)

## 🛠️ Using the Application

Once the app opens, you will see three main sections:

- **Fixable Items:** Tasks and parts ready for repair or replacement now.
- **Blocked Items:** Issues that await approval or missing parts.
- **Audit Log:** A history of all actions and decisions made in the system. Useful for tracking changes.

You can click any item to see detailed information. The app highlights the reasons behind delays or approval holds.

The AI suggestions appear next to some items but require manual confirmation to proceed.

## 🔐 Controlled Information Access

IX-Sustainment-OS limits who can see certain technical data. Access depends on your user role. Common roles include:

- **Supervisor:** Can approve AI recommendations and access full data.
